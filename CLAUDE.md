This is MT Truck & Trailer Repair's shop management app (Burguete Enterprise Inc., Adelanto CA). Owner: Guillermo, sole decision-maker, not a professional developer — explain changes in plain language.
ARCHITECTURE: shop.html is a single-file PWA — all HTML, CSS, and JS in one file. Hosted on GitHub Pages, live at mttruckandtrailerrepair.com/shop.html. Data persistence via Supabase. All Anthropic API calls go through a Cloudflare Worker (mt-ai-proxy) — never put API keys in the browser code. SMS via Twilio (+1 909-550-5331). Email via EmailJS. The public site (index.html, city pages, privacy.html, sms-terms.html) is in this same repo.
CODE CONVENTIONS: ES5-style JavaScript (var, function(){}), no build step, no frameworks. Bilingual EN/ES — all user-facing strings go through the t() function with keys in the TR object; do not use accented characters in ES strings (use \u escapes like \u00F1 if needed). UI follows an Apple-like style: #185FA5 blue, rounded cards, 0.5px borders. Data arrays (WO, INV, CUST, etc.) are saved via saveAll() — always call it after changing data.
RULES: Always syntax-check the JS before committing. Never remove existing features. Deploying = push to main (GitHub Pages auto-publishes in ~1 minute). Roles: owner (PIN), office staff, tech (Tech Portal PINs).
